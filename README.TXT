Welcome to NEO OS!

This is a simple 32-bit operating system written from scratch.

Features:
- VGA text output
- Keyboard input
- Memory management
- Interrupt handling
- Simple shell
- FAT12 file system support

Commands available:
- help: Display available commands
- clear: Clear the screen
- echo: Print text
- about: Display OS info
- meminfo: Display memory info
- time: Display tick count
- ls: List files
- cat: Display file contents
